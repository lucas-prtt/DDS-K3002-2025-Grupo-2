¿Qué es MetaMapa?

MetaMapa es una plataforma de código abierto diseñada para el mapeo colaborativo de información. Nuestro objetivo es maximizar la disponibilidad y veracidad de la información relevante, mientras protegemos la identidad y privacidad de quienes nos visitan.

A través de un sistema distribuido de fuentes de datos, criterios de consenso y protección de identidades, MetaMapa permite a usuarios y organizaciones consultar, contribuir y gestionar información geolocalizada de manera segura y confiable.

Características Principales

Protección de Identidad

Sistema robusto de identidades digitales que permite a los usuarios contribuir información de forma anónima o con diferentes niveles de revelación de identidad.

Colecciones Temáticas

Organización de información en colecciones especializadas con criterios de pertenencia personalizables y algoritmos de consenso para validar la veracidad de los hechos.

Mapeo Geográfico

Visualización interactiva de hechos geolocalizados en un mapa colaborativo, permitiendo explorar información contextualizada por ubicación geográfica.

Fuentes Múltiples

Integración de fuentes estáticas (archivos CSV), dinámicas (APIs) y proxy para agregar y validar información de múltiples orígenes.

Arquitectura del Sistema

MetaMapa está construido con una arquitectura de microservicios que garantiza escalabilidad, mantenibilidad y separación de responsabilidades:

  • • Agregador: Núcleo central que gestiona colecciones, hechos y usuarios
  • • API Pública: Interfaz para consultas públicas sin autenticación
  • • API Administrativa: Gestión de usuarios, identidades y solicitudes
  • • Fuentes de Datos: Módulos especializados para diferentes tipos de fuentes
  • • Dashboard: Panel de visualización y estadísticas
